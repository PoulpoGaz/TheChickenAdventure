## THE CHICKEN ADVENTURE I

# 1) Installation

* Téléchargez, installez et lancez TI-Connect CE
* Transférez sur votre calculatrice les bibliothèques C: https://github.com/CE-Programming/libraries/releases/latest
* Transférez CHKNADV.8xp

# II) Lancement

Choisissez la commande Asm( dans le menu catalogue ([2nd][0]) et séléctionnez CHKNADV dans le menu des programmes. Faites Entrer.

# III) Contrôle

[Flèches directionnelles] - Se déplacer
[2nd]                     - Recharger le niveau en cours
[del]                     - Quitter le niveau
[clear]                   - Quitter le jeu

# IV) Code source

Github: https://github.com/PoulpoGaz/TheChickenAdventureI

# V) Info suplémentaire
